## 微信聊天记录搜索 (WeChat Search)

可以搜索用户的微信聊天记录，支持多账号。

**使用场景：**
- 「我之前和谁聊过XXX？」
- 「找一下关于XXX的聊天记录」
- 「XX说的那个地址/链接是什么？」
- 「越媛说的那个餐厅叫什么？」

**命令：**
```bash
wechat                               # 显示账号列表和帮助
wechat account                       # 显示所有账号
wechat search "关键词"               # 搜索所有账号
wechat search --account 李老师 "关键词"  # 搜索特定账号
wechat search --chats                # 列出所有聊天
wechat search --chats --account 李   # 列出特定账号的聊天
```

**搜索语法：**
```bash
"托福"                    # 简单关键词
"越媛: 学习"              # 搜索特定联系人的聊天（冒号分隔）
"托福 写作"               # OR 逻辑（任一关键词）
"+托福 +写作"             # AND 逻辑（两个都要）
```

**快速参考：**
| 用户请求 | 命令 |
|---------|------|
| 找关于X的消息 | `search "X"` |
| 李老师账号里找X | `search --account 李老师 "X"` |
| 越媛说的关于X | `search "越媛: X"` |
| 列出所有聊天 | `search --chats` |
| 有哪些账号 | `account` |

**注意：**
- 如果用户要「最新」消息，先运行 `wechat sync` 再搜索
- `--account` 支持部分匹配（如「李」可匹配「李老师」）

- **日历**: 查看日程安排

- **视频会议分享**: 修改录像分享权限需要用户授权
