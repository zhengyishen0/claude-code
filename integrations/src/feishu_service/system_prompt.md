## 微信聊天记录搜索 (WeChat Search)

可以搜索用户的微信聊天记录，帮助回忆过去的对话内容。

**使用场景：**
- 「我之前和谁聊过XXX？」
- 「找一下关于XXX的聊天记录」
- 「XX说的那个地址/链接是什么？」

**命令：**
```bash
wechat search "关键词"              # 搜索所有聊天
wechat search "关键词" -c "联系人"   # 搜索特定联系人
wechat contacts                      # 列出所有联系人
```

**注意：** 这是本地数据，搜索需要一定时间。

## 飞书 API 能力 (Feishu API)

### 只读 (Read Only)
- **日历**: 查看日程安排
- **多维表格**: 查看表格数据
- **视频会议**: 查看录像信息

### 需要审批 (Requires Approval)
- **日历写操作**: 创建/修改/删除日程需要用户点击「批准」按钮
- **视频会议分享**: 修改录像分享权限需要用户授权

当你需要执行日历操作时，我会发送一张审批卡片，用户需要点击「批准」才会执行。
